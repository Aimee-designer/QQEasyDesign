使用文档

产品介绍

平台概述

QQ宝藏 EasyDesign 是一个专业的在线图片设计平台，专为宣发内容制作而设计。平台支持两种主要设计模式：

  • 公众号长图模式：适用于微信公众号文章配图、朋友圈分享等场景
  • 小红书多图模式：专为小红书笔记发布优化的多页设计

核心特性

  • 双模式设计 — 一个平台，两种专业模式
  • 丰富的内容类型 — 文字、图片、视频、动画、表情包等
  • 完整的模板系统 — 保存、加载、导入导出模板
  • 精准的导出控制 — 多分辨率、分段切图、批量下载
  • 现代化界面 — 支持日夜间主题切换
  • 数据持久化 — 本地存储，数据不丢失

适用场景

使用场景推荐模式输出规格
微信公众号文章配图公众号长图375px宽长图

产品宣传长图公众号长图支持分段切图
活动推广素材两种模式均可根据平台需求选择

快速入门

第一次使用

  1. 打开平台：在浏览器中打开 QQEasyDesign_v2.html
  2. 选择模式：点击顶部的"公众号长图"或"小红书多图"按钮
  3. 开始设计：点击画布中的"+ 添加新段落"开始创作

基础操作流程

选择设计模式→ 添加内容段落→ 编辑文字和媒体→ 调整样式→ 预览效果→ 导出图片

5分钟快速上手

步骤1：创建第一个段落

  • 点击画布底部的"+ 添加新段落"按钮
  • 选择"模块标题"类型
  • 输入标题文字，如"产品介绍"

步骤2：添加正文内容

  • 再次点击"+ 添加新段落"
  • 选择"正文标准"类型
  • 输入产品描述文字

步骤3：插入图片

  • 点击"+ 添加新段落"
  • 选择"媒体模块"类型
  • 拖拽图片文件到上传区域或点击上传

步骤4：导出图片

  • 点击右上角"导出长图"按钮
  • 选择分辨率（推荐1125 @3x）
  • 点击"导出全图"完成

内容编辑功能

文字编辑

文字类型

类型样式特点字号适用场景
模块标题带装饰线的标题样式18px章节标题、模块分隔
正文大字粗体大号文字20px重点内容、小标题
正文标准标准正文样式16px主要内容、详细描述

文字格式化

  • 加粗：点击工具栏中的 B 按钮
  • 颜色设置：QQ品牌蓝(#0099FF)、警示红色、默认黑色
  • 高亮文字：使用 [需要高亮的文字] 语法，自动转换为黄色背景高亮

实时编辑

  • 点击文字即可直接修改
  • 按 Enter 键换行
  • 支持混合格式富文本

图片编辑

支持的媒体格式

  • 图片：JPEG、PNG、SVG、GIF、WebP
  • 视频：MP4、WebM
  • 动画：Lottie JSON 文件

上传方式

  1. 拖拽上传：直接拖拽文件到上传区域
  2. 点击上传：点击上传区域选择文件
  3. 粘贴上传：Ctrl/Cmd+V 粘贴剪贴板图片

图片调整

  • 使用滑块调整宽度（10%-100%）
  • 自动保持宽高比例
  • 显示媒体类型标签和文件信息

特殊组件

搜索引导组件

  • 模拟QQ搜索框样式，引导用户搜索特定内容
  • 可编辑搜索词、标语文字和主题色

手机预览组件

  • 展示手机界面截图，支持高亮区域标注
  • 操作：上传截图 → 选择高亮区域 → 调整框的位置和大小
  • 支持拖拽移动、尺寸调节、多区域标注

表情包组件

  • 内置丰富的表情包资源，4×4网格布局
  • 一键选择应用，支持大小调整

样式控制

  • 元素可见性：控制元素在导出中的显示/隐藏状态
  • 段落排序：支持上移下移和拖拽排序
  • 颜色标识：5种颜色标记不同段落类型

模板管理系统

模板保存

保存为普通模板

  1. 完成设计内容
  2. 点击顶部"模版管理"下拉菜单
  3. 选择"保存模板"
  4. 输入模板名称并确认
普通模板保存设计结构和样式，不包含媒体文件，存储在浏览器本地，加载速度快。

保存为预制模板

  • 在段落管理面板中点击"固定为预制"按钮
  • 包含完整媒体数据，存储在IndexedDB中

模板加载

模板分类

  • 内置模板：系统预设的专业模板
  • 用户模板：用户自己保存的模板，支持重命名和删除
  • 预制模板：包含媒体数据的完整模板

加载操作

  1. 点击"加载模板"按钮
  2. 在弹窗中浏览并选择模板
  3. 点击模板卡片加载
  4. 确认替换当前设计

模板导入导出

导出模板

  • 在模板列表中点击"导出"按钮
  • 导出为 .edtpl.json 格式，包含完整设计数据和媒体文件
  • 适用于模板备份、团队分享、跨设备同步

导入模板

  1. 点击"导入模板文件"按钮
  2. 选择 .json 或 .edtpl.json 格式文件
  3. 系统自动解析验证并添加到模板库

管理技巧

  • 命名规范：按用途/风格/尺寸分类，如 产品介绍_v1、简约风格_蓝色
  • 版本控制：在名称中标注版本号，保留关键版本
  • 团队协作：设计师创建模板 → 导出文件 → 分享给团队 → 导入使用

常见问题解答

使用问题

Q1: 图片上传失败？

  • 确保使用 JPEG、PNG、GIF、SVG 或 WebP 格式
  • 单个文件建议不超过10MB
  • 推荐使用 Chrome、Firefox 或 Safari 浏览器

Q2: 导出图片不清晰？

  • 选择 1125px @3x 高分辨率
  • 确保上传的原图本身清晰
  • 避免过度缩放小图片

Q3: 模板保存后找不到？

  • 确认看到"保存成功"提示
  • 检查"用户模板"或"预制模板"分类
  • 确保浏览器允许本地存储

Q4: 小红书多图段落显示不全？

  • 确保段落已正确分配到页面
  • 单页内容不要过多，合理使用分页
  • 检查元素可见性设置

技术问题

Q5: 浏览器提示内存不足？

  • 减少段落数量，压缩大图片
  • 关闭其他标签页，重启浏览器

Q6: 导出卡住不动？

  • 复杂设计需要更多处理时间，请耐心等待
  • 尝试分段导出或刷新后重试

Q7: 模板导入失败？

  • 确保文件为 .json 或 .edtpl.json 格式
  • 确保文件完整、JSON格式正确

数据安全

Q8: 设计数据会丢失吗？

  • 数据存储在本地浏览器中，不会上传到服务器
  • 清除浏览器数据会删除设计，建议定期导出模板备份
  • 将 .edtpl.json 文件保存到云盘作为备份

兼容性

Q9: 推荐浏览器？

  • 推荐：Chrome 80+、Firefox 75+、Safari 13+
  • 避免：IE浏览器、过旧版本浏览器
  • 建议在桌面端进行设计，移动端可预览效果
